# Screenshots

Drop your platform screenshots here. The main `README.md` already references these four files:

| File name | What to capture |
| --- | --- |
| `tutor.png` | The AI tutor answering a question (with a rendered formula) |
| `quiz.png` | A dynamic quiz + the error-analysis / feedback panel |
| `graph.png` | The Neo4j knowledge graph (concepts & prerequisites) |
| `guided.png` | An interactive SVG "guided example" in action |

## How to add them

1. Save your captures in this folder with the **exact names** above.
2. In the main `README.md`, replace the italic placeholders (e.g. `_`docs/screenshots/tutor.png`_`) with real image tags:

   ```markdown
   ![Adaptive Tutor](docs/screenshots/tutor.png)
   ```

## Tips for clean captures

- Use a width around **1280–1600 px** so images stay sharp on GitHub.
- Prefer **light, uncluttered** screens; hide personal data.
- Keep the four images visually consistent (same browser zoom / theme).
- Optional: a short **demo GIF** (`demo.gif`) at the top of the README is very effective.
